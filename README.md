# 📚 StudyAI — AI-Powered Smart Study Platform

StudyAI is an all-in-one, intelligent study companion app designed to help students learn faster, study smarter, and track their academic progress. Powered by Groq's high-speed **LLaMA 3.3** model, the platform parses uploaded study documents to generate comprehensive summaries, interactive custom quizzes, dynamic digital flashcards, personalized 7-day study schedules, and offers an AI study coach with conversational memory.

A sleek, modern dark-themed user interface styled with custom CSS provides responsive navigation, glassmorphism card panels, custom toast notifications, and interactive charts.

---

## ✨ Features

- **📄 Multi-Format Document Parser**: Seamlessly upload files in PDF, DOCX, or TXT formats, or paste text manually, and let StudyAI extract and analyze the content.
- **📝 Intelligent Summary Generator**: Generates comprehensive summaries with key takeaways from your materials. Export generated study guides as clean PDFs with a single click.
- **🧠 Custom Quiz Engine**:
  - Multiple Choice (MCQ), True/False, and Short Answer question generation.
  - Configurable quiz difficulty (`easy`, `medium`, `hard`) and number of questions.
  - Interactive test runner with active countdown timers.
  - Real-time AI self-grading with feedback and model answers for short-answer inputs.
- **🗂️ Interactive Flashcard Decks**: Automatically converts study text into digital flashcards with an elegant card-flipping UI, progress tagging (e.g., marking cards as "learned"), and search functionality.
- **📅 7-Day Study Planner**: Generates a tailored study plan structured across a week based on your material's content.
- **📊 Analytics & Performance Insights**:
  - Live study stats dashboard tracking streaks, total study hours, and card retention.
  - Visual charts showing weekly study time and quiz performance trends (using Chart.js).
  - Identification of weak topics with AI-generated feedback and insights on how to improve.
- **💬 Personal AI Study Coach**: Ask follow-up questions about your files or chat freely. The AI keeps track of the context and even recommends dynamic follow-up queries.
- **💾 Dual Database Engine (Hybrid Storage)**: Use remote Firebase Firestore for secure cloud sync, or let it automatically fall back to local filesystem storage (JSON files under `data/`) for zero-configuration, instant usage.
- **🔐 User Access Management**: Secure login and signup views to persist personalized user dashboards.

## 🛠️ Tech Stack

### Frontend
- **Framework**: [React 18](https://react.dev/) (Vite bundler)
- **Routing**: [React Router DOM v7](https://reactrouter.com/)
- **Charts**: [Chart.js](https://www.chartjs.org/) & [React-Chartjs-2](https://react-chartjs-2.js.org/)
- **PDF Generation**: [jsPDF](https://github.com/parallax/jsPDF)
- **Styling**: Vanilla CSS (Modern dark mode, responsive grid systems, smooth micro-interactions, glassmorphic shadows)

### Backend
- **Framework**: [Flask](https://flask.palletsprojects.com/) (Python)
- **AI Integration**: [Groq SDK](https://github.com/groq/groq-python) (running `llama-3.3-70b-versatile`)
- **Document Extractors**: `pypdf` (for PDF processing) and `python-docx` (for MS Word files)
- **Database Options**: Firebase Admin SDK (Cloud Firestore) & Native Python JSON engine

## 🚀 Getting Started

### Prerequisites
Before setting up the project, make sure you have the following installed:
- [Python 3.8+](https://www.python.org/downloads/)
- [Node.js v16+](https://nodejs.org/)
- A **Groq API Key** (Get one for free at the [Groq Console](https://console.groq.com/))

---

### Step 1: Clone and Prepare Workspace
Clone this repository to your local machine and navigate into the root directory:
```bash
git clone https://github.com/your-username/studyai-project.git
cd studyai-project
```

### Step 2: Configure Environment Variables
Create a `.env` file in the **root** directory (you can copy the provided example):
```bash
cp .env.example .env
```
Open `.env` and fill in your Groq API key:
```env
# Groq API Configuration
GROQ_API_KEY=gsk_your_groq_api_key_here

# Flask server configuration
FLASK_PORT=5000
FLASK_DEBUG=True
```

*(Optional)* **Firebase Setup**: If you want to use Firebase Firestore, place your Firebase Service Account JSON credentials file named `firebase-credentials.json` in the root folder. If not present, the server will automatically default to local file storage under `data/`.

---

### Step 3: Run the Python Backend
1. Create a Python virtual environment:
   ```bash
   # Windows
   python -m venv venv
   .\venv\Scripts\activate

   # macOS/Linux
   python3 -m venv venv
   source venv/bin/activate
   ```
2. Install Python packages:
   ```bash
   pip install -r requirements.txt
   ```
3. Start the Flask development server:
   ```bash
   python app.py
   ```
The backend server will run on [http://localhost:5000](http://localhost:5000).

---

### Step 4: Run the React Frontend
1. Open a new terminal tab and navigate into the `frontend` folder:
   ```bash
   cd frontend
   ```
2. Install Node.js packages:
   ```bash
   npm install
   ```
3. Start the Vite React development server:
   ```bash
   npm run dev
   ```
4. Click the URL displayed in your terminal (usually [http://localhost:5173](http://localhost:5173)) to launch StudyAI in your browser.

## 💡 Usage Guide

1. **Sign Up**: Start the frontend and sign up for a new account.
2. **Upload Materials**: Go to the **Upload Material** tab. Drop a PDF/DOCX/TXT file or paste some study notes, and click **Upload**.
3. **Summarize**: Navigate to the **Summary** tab to read an AI-structured version of your material. Click **Download Summary PDF** to save it locally.
4. **Take Quizzes**: Open the **Quiz** tab, select your difficulty level and format, then start the quiz. Complete the timer-based test and view your results instantly!
5. **Flip Flashcards**: Browse the **Flashcards** tab. Click cards to flip between term/definition, mark them as learned, or search through terms.
6. **Track Progress**: Head to **Analytics** to see your study time chart, performance trend line, streaks, and view custom recommendations written by LLaMA 3.3.
7. **Ask AI Coach**: Go to the **AI Coach** tab to chat with the LLaMA 3.3 coach about any concept in your notes.
